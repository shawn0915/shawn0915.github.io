不过，为了确保用户安全，尤其是 root 用户密码的健壮性，MySQL 服务器默认启用了密码校验组件。

validate_password 组件

是的，这里的表述是“组件”，不再是“插件”。

从 MySQL 8.0 开始，MySQL Server 引入了基于组件的基础架构，用于提高服务器的可扩展性。

相关参数更名为以 validate_password.* 前缀。
