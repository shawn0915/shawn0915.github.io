我们很高兴的宣布，在 新发版模型 下，MariaDB第三代系列–MariaDB 10.9.0发布。

MariaDB成熟地更快

新模型的目标之一，是将特性更快的交付给用户，在新特性更容易的被测试、同时开发多个新特性的情况下，保证稳定性。

MariaDB 10.9 的候选功能

有四个不同的预览版本，分别包含下面的特性，并考虑在10.9.1发布：

SHOW ANALYZE FORMAT=JSON

  1. 扩展 SHOW EXPLAIN 命令，以支持 SHOW ANALYZE [FORMAT=JSON] (MDEV-27021 )
  2. 增加 EXPLAIN FOR CONNECTION 语法，以支持 SHOW EXPLAIN (MDEV-10000 )

JSON Range 表达式

  1. 为JSON Path实现Range表达式 (MDEV-27911 )
  2. 支持JSON Path负索引 (MDEV-22224 )
  3. JSON_OVERLAPS方法 (MDEV-27677 )

重做日志异步写

  1. 重做日志异步写(MDEV-26603 )

其他项

  1. 实现 mysqlbinlog的 –-do-domain-ids, –-ignore-domain-ids, –-ignore-server-ids 选项(MDEV-20119 )
  2. information_schema.tables.table_type 现在将本地临时表显示为TEMPORARY (MDEV-12459 )
  3. 合并“old”SQL变量到“old_mode” (MDEV-24920 )
  4. 保险密钥管理插件 (MDEV-19281 )
  5. wsrep解决状态，SST进程日志的JSON文件接口 (MDEV-26971 )
  6. 允许不重启数据库改变innodb_log_file_size (MDEV-27812 )

在接下来的几天里，将单独发博客说明这些特性，但在此期间，您可以在 发版说明 中阅读更多内容。

在 10.9.0的下载页面 上，您将看到Linux（x86-64 bintar）和源码两种格式。

这些特性将在通过测试后合并进 MariaDB 10.9.1。

我们期待收到您的评论！
